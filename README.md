# GradeExplorer
## Eduardo Liñares Fernández

Proyecto para la asignatura Desarrollo de Interfaces, de 2º de DAM.

---
La aplicacion permitirá gestionar un pequeño grupo de alumnos (una clase), mostrando, y permitiendo editar su información.
Está dividida en vistas, a las que se accede desde el menú lateral izquierdo.
